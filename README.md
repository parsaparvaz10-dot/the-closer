# The Closer — AI Deal Evaluation Agent

**Motayo Investments** · Confidential — Internal Use Only

An AI-powered property flip evaluation agent for the Houston, TX market. Runs a 5-step agentic pipeline to produce a scored deal memo with BUY / NEGOTIATE / PASS recommendation.

## Pipeline

1. Property Intel — tax records, Zillow, Redfin, HAR
2. Comp Analysis — comparable sales, ARV range
3. Condition & Risk — 9 building systems + Houston flood/soil
4. Rehab Estimation — three-tier Houston contractor pricing
5. Deal Scoring — profit models, deal score (1-100), verdict

## Usage

Runs as a React JSX artifact inside Claude.ai (Pro/Team subscription required). Paste the-closer-agent.jsx into a new artifact.
